7.- Indica què ens aporta Samba que no tingui ja NFSv3.

La diferencia es que samba ens permet treballar amb usuaris, mentre que nfs únicament ens permet treballar amb xarxes.

8.- Quina diferència hi ha entre Samba i CIFS?

CIFS forma parte del protocolo smb y permite conectar de forma remota archivos y datos de windows con linux o mac.

En cambia samba es una implementación de cifs i smb pero con código abierto.

9.- Quins serveis has utilitzat a Samba? Per a què serveix cadascun?

Serveis de compartir carpetes, compartir impressores , donar accés a usuaris, grups i posar un nom al servidor.

10.- Quin port utilitza Samba?

Utilitza el port 445.
